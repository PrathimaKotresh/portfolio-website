Mobile Chat App

A React Native app for mobile devices which provides users with a chat interface to chat and the possibility to share images and their location. The app is optimized for both Android and iOS devices. Expo.io is used to develop the app and Google Firestore is used to store the chat messages.

Technical details:

  • React Native
  • React Native Gifted Chat
  • React Native Maps
  • Expo.io
  • Google Firestore Database for storage
  • Google Firebase authentication
  • React Native AsnyStorage for app local storage
