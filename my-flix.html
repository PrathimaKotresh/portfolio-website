MyFlix - Movie App

Objective: To build a “movies” ReactJS web application using RESTful movies API. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Tech stack:

MERN (MongoDB, Express, React, and Node.js) stack.

Backend Technical details:

  • RESTful API Node.js and Express application
  • Body­parser package for reading data from API requests
  • Morgan for API logging
  • Database using MongoDB
  • Business logic modelled with Mongoose
  • API Testing using Postman
  • API hosted in Heroku
  • Authentication and Authorization using JWT strategy

Frontend Technical details:

  • ReactJS
  • Parcel build tool, live reloading and Babel to transpile
  • API communication using axios
  • Routing using react-router-dom
  • React Redux
  • App hosted in Heroku
