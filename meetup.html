Meetup App

Objective: To build a serverless “meetup upcoming events for a selected city” ReactJS progressive web application using using a test-driven development (TDD) technique. The application uses the Meetup API to fetch upcoming events for a selected city.

Technical details:

  • ReactJS
  • App hosted in gh-pages
  • API communication using axios
  • Serverless functions for authorization server using AWS lambda
  • An alert system using an OOP approach
  • Unit Testing and Integration Testing using Jest, enzyme
  • User acceptance testing using jest-cucumber
  • E2E testing using puppeteer
  • Application performance monitoring using Atatus, integrated into app using atatus-js
  • Conversion of app into Progressive Web App (PWA) using Lighthouse tool
  • Data Visualization using Recharts
