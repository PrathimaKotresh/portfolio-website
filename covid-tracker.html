Covid-19 Tracker

Technical details:

  • JavaScript
  • CSS
  • Functional programming
  • DOM interactions
  • Event handling
  • Ajax and Fetch to interact with an External API
  • Polyfills to mimic newer JavaScript features for older browsers
  • Responsive modals, navbar and grid using Bootstrap
  • jQuery
